Angular Ngxecharts Baidu Bmap

2021-04-07 11 分钟阅读时间 Angular, ECharts

@TOC

安装插件

npm install echarts
npm install ngx-echarts

配置ngx-echarts

import { NgxEchartsModule } from 'ngx-echarts';

@NgModule({
  imports: [
    ...,
    NgxEchartsModule
  ],
})
export class AppModule { }

引入百度地图

申请百度地图密钥

申请链接：http://lbsyun.baidu.com/index.php?title=jspopular/guide/getkey） 记住ak码，在浏览器端使用，请选择“浏览器端AK”，例如JavaScript API只支持浏览器类型AK。

在index.html中引入

<script type="text/javascript" src="https://api.map.baidu.com/api?v=2.0&ak=xxxxxxxxxxxxx"></script>

注：百度地图必须在根组件index.html页面引入，否则会报错

  convertData(data: any) {
    var res = [];
    for (var i = 0; i < data.length; i++) {
      var geoCoord = this.geoCoordMap[data[i].name];
      if (geoCoord) {
        res.push({
          name: data[i].name,
          value: geoCoord.concat(data[i].value)
        });
      }
    }
    return res;
  }
